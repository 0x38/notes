#Notes

![Notes app](https://sii.im/playground/notes/notes-screenshot-github.png "Notes app")

A simple, offline-based notepad, based on using [Service Worker](https://developer.mozilla.org/en-US/docs/Web/API/Service_Worker_API) and [PWA techniques.](https://www.smashingmagazine.com/2016/08/a-beginners-guide-to-progressive-web-apps/)

View the demo [here](https://sii.im/playground/notes) on a [capable device.](http://caniuse.com/#feat=serviceworkers)

## Features

  - Write notes and save them to [localStorage](https://developer.mozilla.org/en/docs/Web/API/Window/localStorage)
  - Use Service Worker to enable offline viewing
  - "Add To Home Screen" feature on Android supported devices
